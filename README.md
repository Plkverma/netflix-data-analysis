# Netflix Data Analysis Project

## Overview
This project performs exploratory data analysis (EDA) on Netflix content to understand trends in movies and TV shows.

## Key Questions
- Is Netflix focusing more on TV shows or movies?
- How has the average movie duration changed over time?

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- TV shows have increased more rapidly than movies in recent years.
- Average movie duration has gradually increased over time.

## Dataset
Netflix Movies and TV Shows dataset from Kaggle.
